# 📖 About

**SouqHub** is a modern and responsive fashion e-commerce web application designed to deliver a clean and engaging online shopping experience.

The project showcases reusable React components, responsive layouts, dark/light mode support, shopping cart functionality, product search, and modern frontend development practices using React.js, Vite, and Tailwind CSS.

- 🌙 Dark & Light Mode
- 🛒 Shopping Cart

- ⭐ Top Rated Products

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/minaasamer/SouqHub.git
```

Go to the project folder

```bash
cd SouqHub
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
